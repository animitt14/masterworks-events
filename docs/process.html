RSVP Dashboard Process

1 · Pull RSVPs from AdEvents

1
Go to Redash query #11069. Set the Date parameter to the event date (YYYY-MM-DD) and run the query.
2
Download the results as a CSV.

2 · Import to HubSpot

1
In HubSpot, go to Contacts → Import. Upload the CSV.
2
Map columns to HubSpot properties. Where prompted, check "Don't override existing values" to avoid overwriting any data already in HubSpot.
3
Set outbound_rsvp_to_event to the event date for all imported contacts.

3 · Enrichment

After import, contacts often have no job title or company. There are two enrichment layers:

Auto 8am daily rebuild

The script searches LinkedIn, company pages, and other public sources to find title/company for each blank contact (up to 95 enrichment calls/day). Work email domains are used as a company fallback when no other data is found.

Auto 8:30am enrichment routine

A separate scheduled agent runs after the rebuild. It web-searches today's RSVP contacts still missing title/company (LinkedIn, ZoomInfo, company pages), patches HubSpot directly, and does a second pass on anyone still blank. Runs every day at 8:30am ET.

Manual Ad-hoc enrichment

Open Claude Code and say "enrich today's names" (or the specific event date). Claude will search each remaining blank, patch HubSpot, and rebuild the dashboard. Always does two passes — one broad search, then a second pass on anyone still unresolved.

4 · Scoring

Every contact gets a score of 1–5 computed automatically from title, company, email domain, estimated net worth, and lifecycle stage.

  • 5 (High) — Finance domain, physician, C-suite at named top-tier firm, already invested or Opportunity stage
  • 4 (Medium-High) — VP / Director level, finance firm without top-tier signal
  • 3 (Medium) — Senior manager, attorney, mid-tier signals
  • 2 (Low-Medium) — Founders (unverified), analysts, no-shows, no data
  • 1 (Low) — Wealth advisors (refer, don't invest), art world, real estate agents, not interested

To override a score for a specific contact, edit SCORE_OVERRIDES in generate_rsvp.py — add the HubSpot contact ID and the desired score (1–5). Changes go live on the next rebuild.

5 · Uninviting

The dashboard has an uninvite button per contact. Clicking it writes to a shared Gist. The next rebuild syncs those uninvites to HubSpot (sets outbound_event_attendee_disqualified) and clears the Gist. To uninvite in bulk or outside the dashboard, set the property directly in HubSpot.

6 · Pipeline & Deal Management

Deals live in the Gallery Leads pipeline in HubSpot. After an event:

  • Move attended contacts from Event Attended → Contacted once you've followed up
  • Log calls and meetings as activities on the contact record
  • Create a deal if a contact expresses interest; link it to the contact
  • The pipeline dashboard (separate from this one) shows Group 1 contacts with deal stage, last contact date, tasks, and LinkedIn

To generate deal amount predictions for Gallery Leads, open Claude Code and ask: "run deal amount predictions." Claude will match deals to associated contacts and output an HTML table with conservative estimates.

7 · The GitHub Script

The dashboard is generated by generate_rsvp.py in the animitt14/masterworks-events repo. You don't need to edit it for day-to-day use.

  • Adding a score override — find SCORE_OVERRIDES near the top of the script and add 'contact_id': score
  • Adding a team member — find the OWNERS dict and add their HubSpot owner ID and name
  • Deploying changes — commit and push to main; GitHub Actions rebuilds the dashboard automatically within a few minutes
  • Don't edit docs/index.html directly — it gets overwritten on every rebuild

For anything more involved, open Claude Code in the repo directory and describe the change — it knows the full codebase.

8 · Automated Processes (reference)

  • 8:00am ET Full rebuild — fetches all RSVPs (180 days back, 30 days ahead), runs Google enrichment (~95 contacts/day), PLUTO property value lookups for NYC addresses, RocketReach photo enrichment, domain-fallback company fill, outputs and pushes index.html
  • 8:30am ET Enrichment routine — web-searches today's RSVP blanks (title/company), patches HubSpot, two-pass search
  • On rebuild Uninvite sync — reads shared Gist, sets disqualified flag in HubSpot, clears Gist
  • On rebuild Send-confirmation sync — marks contacts as confirmed if confirmation email was sent

All automated jobs are managed via GitHub Actions (.github/workflows/daily.yml) and the Claude Code scheduled routine. View scheduled routines at claude.ai/code/routines.
